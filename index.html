Leaf Disease Detection App

Our project aims to assist farmers in detecting plant diseases in their crops at an early stage, while also suggesting suitable crops for cultivation based on soil attributes. We have implemented a deep learning algorithm using CNN (Convolutional Neural Networks) to predict leaf diseases accurately.

Djnago Blog App

Blog application built using Django, a powerful web framework for Python. The app provides a platform for users to create and publish blog posts. One of the key features of our blog app is the use of the Django admin panel, which provides an intuitive interface for managing the app's data.

GitHub Finder




The GitHub Finder web application is a platform that allows users to search and view GitHub profiles of developers. This project is built using the React JavaScript library,which is widely used for building user interfaces. When a user enters a username, the application makes an API call to the GitHub API to retrieve the user's profile information.
